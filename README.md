# Cordova-Plugin-TI-WIFI
It provides searching the ssid of connected wifi and setting the ssid to TI wifi modules on the android's phones.
# 功能
 封装了TI WIFI模块提供的smartconfig部分功能
- 搜索当前连接的WIFI路由器的SSID
- 将SSID和密码设置到TI WIFI模块
- 搜索已经配置好SSID的设备

# API
- getssid
- startconfig
- stopconfig
- startfinddevice
- stopfinddevice
